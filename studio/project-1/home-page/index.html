              ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                                                        
                                                   
                                                            
                                          ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                       ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ     ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                 ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ  
ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ 

                            ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                                                                  ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ 
ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                                                                  ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ 
               ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ 
                                                            ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                          ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ 
ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                  ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                                                 ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ 


                   ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                 ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                 ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                 ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ 


ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                                                                 ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ                         ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ ᵒⁿ ᵖᵘⁿᶜᵗᵘᵃᵗⁱᵒⁿ
